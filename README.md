# monkey-server

*Ye Olde install'r Mix-n-Mojo*

## Install

You need GIT, duh, then clone in your root folder

```bash
su
cd
apt-get install git
git clone https://github.com/gdripoll/monkey-server.git
cd monkey-server
```

Ready!

## Use

Inform the `user` to use in any installation

```bash
USERNAME=goose
```
